如何安装及使用HEXO博客框架

欢迎来到我的第一篇博客，这里我将介绍如何用hexo专业博客框架进行博客的撰写。

1.下载安装Node.js及Git

Node.js：https://git-scm.com or https://git-scm.com
Git：https://nodejs.org/en/ or https://git-scm.com

2.打开 Git Bash,接下来的命令均在Git Bash中执行

安装 Hexo ：$npm install -g hexo-cli
新建博客文件夹：$hexo init blog
定位到该文件：$cd blog
安装依赖包： $npm install
打开本地服务：$hexo server

. blog 文件夹目录
├── _config.yml 网站的 配置 信息
├── package.json 应用程序的信息
├── scaffolds 模版 文件夹
├── source 资源文件夹是存放用户资源的地方
| ├── _drafts
| └── _posts
└── themes 主题 文件夹

3.测试和配置

  1. 测试：打开 Git Bush Here，输入 git –version，出现版本号则说明 Git 环境配置成功
  2. 配置本机的Git：
    $ git config –global user.name “你的名字”
    $ git config –global user.email “你的邮箱”
  3. 生成密匙：$ ssh-keygen -t rsa -C “邮箱同上”
  4. 复制密匙：在ｃ盘用户.ssh文件下用文本编辑器打开id_rsa.pub并复制文件的内容。
  5. 提交公钥：在github官网上，点击用户下Settings的SSH and GPG keys下New SSH key添加之前的密匙复制进内容就可以了。
  6. 连接完成测试：$ssh git@github.com

4.配置网站信息 _config.yml

配置详情网站：https://hexo.io/zh-cn/docs/configuration.html

theme 当前主题名称。值为false时禁用主题
deploy 部署部分的设置

github获取ssh密钥

deploy:
type: git
repository: git@github.com:microyan/microyan.github.io.git
branch: master

5.指令

  1. $npm update hexo -g #升级

  2. $hexo init #初始化

  3. hexo new “文章名” == hexo n “文章名” #新建文章

  4. hexo generate == hexo g #生成本地静态文件
  5. hexo server == hexo s #启动服务预览
  6. hexo deploy == hexo d #部署

  7. hexo clean #清除缓存

  8. hexo new “文章名” #新建文章

  9. hexo new page “pageName” #新建页面

  10. hexo n #写文章

  11. hexo g #生成静态文件
  12. hexo d #部署 可与hexo g合并为 hexo d -g 部署并上传

6.文章撰写

目录：source/_posts/文章名.md

最后，感谢你的阅读，希望能够帮到你！
